JSON Parser in C++
This project is a simple JSON parser implemented in C++. It parses a basic JSON-like dictionary format (key-value pairs) and displays the parsed data.

Input Format
The parser expects a JSON-like dictionary as input. Each key-value pair should:

Have string keys enclosed in double quotes ("key").
Have string values enclosed in double quotes ("value").
Example Input
{"name":"Rithwik","age":"21","city":"Warangal"}
Output Format
The program displays the parsed key-value pairs line by line.

Example Output
Your Data:
name: Rithwik
age: 21
city: Warangal
Error Handling
If the input format is invalid, the program will print an error message:

Error: Invalid JSON format.
